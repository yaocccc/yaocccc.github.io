Config

布局和使用

左 filetype = dbui
右 filetype = sql
下 filetype = dbout

dbui:
o / <cr> - 打开/关闭节点
R - 刷新
A - 添加链接 (格式见 vim-dadbod-ui 说明)

PS. 如果用户名、密码中有特殊符号 需要进行url encode

sql:
<leader>W 保存当前query到文件

Tips

搭配 coc-sql coc-db 使用，可以对着db、表、字段级别补全
